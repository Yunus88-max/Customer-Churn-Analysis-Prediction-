Project Overview
This project is an interactive Customer Churn Analysis and Prediction Dashboard developed using Power BI. The dashboard analyzes customer data to provide valuable insights into customer behavior, churn patterns, demographics, and service usage, helping businesses improve customer retention and make data-driven decisions.

Features
Churn Rate Analysis
Customer Demographics Insights
Customer Tenure Analysis
Contract Type & Payment Method Analysis
Internet Service Analysis
KPI Tracking
Interactive Filters & Visualizations

Tools & Technologies Used
Power BI
DAX
Power Query
Data Visualization
Business Intelligence
Customer Analytics
Dataset Information

The dataset contains:

Customer Information
Demographic Details
Service Subscription Data
Contract Information
Payment Methods
Monthly Charges and Tenure Data
Churn Status
Dashboard Insights

The dashboard helps in:

Identifying factors influencing customer churn
Analyzing customer demographics and behavior
Monitoring churn rates and retention trends
Understanding the impact of contract types and payment methods
Supporting data-driven business decisions
